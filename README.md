# $\text{Physics-Informed Neural Networks vs 4th Order Runge-Kutta}$ 

Repositori ini membahas mengenai perbandingan antara dua metode pemodelan fisika, yaitu *Physics-Informed Neural Networks* (PINNs) dan *Runge-Kutta Orde 4* (RK4). Untuk menguji kedua metode tersebut, studi kasus yang digunakan adalah *sistem getaran teredam*, dengan tujuan menganalisis akurasi prediksi masing-masing metode terhadap solusi eksak.

## $\text{Latar Belakang}$

Pemodelan sistem fisis dapat dilakukan menggunakan pendekatan numerik klasik maupun pendekatan berbasis pembelajaran mesin:

* *Runge-Kutta Orde 4* (RK4) merupakan metode numerik yang umum digunakan untuk menyelesaikan persamaan diferensial biasa.
* *Physics-Informed Neural Networks* (PINNs) adalah pendekatan berbasis neural network yang mengintegrasikan hukum fisika ke dalam proses pelatihan.

Repository ini bertujuan untuk membandingkan performa kedua metode tersebut dalam memodelkan sistem getaran teredam.
